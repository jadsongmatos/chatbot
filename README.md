# ChatBOT

install dependencies:

```bash
npm install
```

run the app:

```bash
npm start
```

## PYTHON

run the app:

```bash
python -m py_compile app.py
```

uwsgi --ini uwsgi.ini --socket 0.0.0.0:5000 --protocol=http
